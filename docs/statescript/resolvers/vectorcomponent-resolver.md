# VectorComponentResolver

> **Type:** `Gamesmiths.Forge.Statescript.Properties.VectorComponentResolver`
> **Output Type:** `float`

Extracts a single component from a vector.

## Constructor

```csharp
new VectorComponentResolver(vector, component)
```

## Supported Types

| Vector Type | Components |
|-------------|------------|
| `Vector2` | `X`, `Y` |
| `Vector3` | `X`, `Y`, `Z` |
| `Vector4` | `X`, `Y`, `Z`, `W` |
